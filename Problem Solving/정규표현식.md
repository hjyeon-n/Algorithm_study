# 정규표현식

| 정규 표현식                                | 설명         |
| ------------------------------------------ | ------------ |
| ^[0-9]*$                                   | 숫자         |
| ^[a-zA-Z]*$                                | 영문자       |
| ^[가-힣]*$                                 | 한글         |
| \\w+@\\w+\\.\\w+(\\.\\w+)?                 | E-Mail       |
| ^\d{2,3}-\d{3,4}-\d{4}$                    | 전화번호     |
| ^01(?:0\|1\|[6-9])-(?:\d{3}\|\d{4})-\d{4}$ | 휴대전화번호 |
| \d{6} \- [1-4]\d{6}                        | 주민등록번호 |
| ^\d{3}-\d{2}$                              | 우편번호     |
